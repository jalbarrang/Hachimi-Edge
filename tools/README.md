# Dev Tools
These are the tools which can be used when developing this project.

Unless stated otherwise, they're meant to be run in the project's root directory. See each platform's README for more info.

## Refreshing game data

When the global server ships an update, follow
[docs/updating-game-data.md](../docs/updating-game-data.md) to refresh skills,
umas, and support cards (`fetch-master-db`, `skill-grades`, `course-data`,
`tracker-data-manifest`, `gametora-sync`).
